= Arconia Observability
:page-aliases: observation:index.adoc

Arconia extends the observability capabilities of Spring Boot applications in two complementary areas.

xref:opentelemetry:index.adoc[*OpenTelemetry*]. Combines the standardization of https://opentelemetry.io/docs/what-is-opentelemetry/[OpenTelemetry] with the robustness of https://micrometer.io[Micrometer] to deliver a unified solution that covers logs, metrics, and traces. Add a single dependency and get everything you need, with full support for OpenTelemetry APIs, SDKs, and environment variables.

xref:semantic-conventions/index.adoc[*Semantic Conventions*]. A pluggable model for choosing how telemetry data is named and structured. Switch from the default Micrometer conventions to xref:semantic-conventions/opentelemetry.adoc[OpenTelemetry], xref:semantic-conventions/openinference.adoc[OpenInference], xref:semantic-conventions/openlit.adoc[OpenLit], xref:semantic-conventions/openllmetry.adoc[OpenLLMetry], or xref:semantic-conventions/langsmith.adoc[LangSmith] conventions through a single configuration property, with no code changes required.
