= Semantic Conventions
:page-aliases: observation:semantic-conventions/index.adoc

Spring Boot applications instrumented with the Micrometer APIs can export telemetry data adopting specific semantic conventions. Semantic conventions define a standard set of attributes and naming rules for observations, ensuring consistency across different observability backends.

Arconia provides support for the following semantic conventions:

* xref:semantic-conventions/opentelemetry.adoc[OpenTelemetry Semantic Conventions]
* xref:semantic-conventions/openinference.adoc[OpenInference Semantic Conventions]
* xref:semantic-conventions/openllmetry.adoc[OpenLLMetry Semantic Conventions]
* xref:semantic-conventions/langsmith.adoc[LangSmith Semantic Conventions].
