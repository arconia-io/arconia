= Semantic Conventions

Spring Boot applications instrumented with the Micrometer Observation APIs can export telemetry data adopting specific semantic conventions. Semantic conventions define a standard set of attributes and naming rules for observations, ensuring consistency across different observability backends.

Arconia lets you select which semantic conventions to use via the `arconia.observations.conventions.type` configuration property. The default value is `micrometer`, which uses the built-in Micrometer conventions.

.Observation Configuration Properties
|===
|Property |Default |Description
|	`arconia.observations.conventions.type`
|	`micrometer`
|	Type of semantic conventions to use for observations (e.g., `micrometer`, `openinference`).
|===
