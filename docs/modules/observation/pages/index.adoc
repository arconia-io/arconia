= Arconia Observation
:page-aliases: observability:index.adoc

Arconia Observation extends the observability capabilities of Spring Boot applications by providing a pluggable model for xref:semantic-conventions/index.adoc[semantic conventions]. It lets you choose which conventions to adopt when exporting telemetry data, so you can match the expectations of your observability backend.

By default, Spring Boot applications use the built-in Micrometer conventions. Arconia lets you switch to alternative conventions through a single configuration property, with no code changes required.
