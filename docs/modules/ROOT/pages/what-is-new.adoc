= What's New in Arconia {page-component-version}

Arconia {page-component-version} provides a number of new features and enhancements. This page includes the highlights of the release, but you can also check out the full https://github.com/arconia-io/arconia/releases[release notes] for more details about each new feature and bug fix.

We provide an automated way to upgrade your project to the latest version of Arconia, which is described in the xref:upgrading-arconia.adoc[Upgrading Arconia] documentation.

== Dev Services

=== General

* Dev services within the same category are now mutually exclusive. If more than one is active at the same time, the application fails at startup with a clear error message indicating which services are in conflict and how to resolve it. The following groups are enforced:
** *JDBC*: MariaDB, MySQL, Oracle, Oracle XE, PostgreSQL;
** *MongoDB*: MongoDB, MongoDB Atlas;
** *OpenTelemetry*: Grafana LGTM, OpenLit, OpenTelemetry Collector, Arize Phoenix;
** *Redis*: Redis, Valkey.

=== Floci

* A new Floci Dev Service has been introduced providing a https://floci.io[Floci] instance, a free open-source local AWS emulator. It works with https://awspring.io[Spring Cloud AWS]. Learn more about this new Dev Service in the xref:dev-services:floci.adoc[dedicated documentation].

=== OpenLit

* A new OpenLit Dev Service has been introduced providing an https://openlit.io[OpenLit] instance, an open-source AI observability and evaluation platform. It works with Spring AI and xref:opentelemetry:index.adoc[Arconia OpenTelemetry]. Learn more about this new Dev Service in the xref:dev-services:openlit.adoc[dedicated documentation].

== Observability

=== OpenLLMetry Semantic Conventions

* A new `arconia-openllmetry-semantic-conventions` module was introduced, providing support for exporting traces to https://www.traceloop.com[Traceloop] and other AI observability platforms supporting https://traceloop.com/docs/openllmetry[OpenLLMetry] via OpenTelemetry.
* Support for the experimental https://traceloop.com/docs/openllmetry[OpenLLMetry Semantic Conventions for Generative AI], including chat model, embedding model, and tool execution spans; token usage metrics; input/output message content capturing.
