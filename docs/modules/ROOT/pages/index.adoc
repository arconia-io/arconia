== What is Arconia?

Arconia is a framework that acts as an add-on for Spring Boot, designed to enhance modern enterprise applications built with Java. You can add Arconia to an existing Spring Boot application to boost developer experience, reduce boilerplate, and seamlessly adopt cloud native patterns.

NOTE: The project is currently under active development and is working towards its first GA release. Many of the core features are already available. We're working hard to improve it and appreciate your patience as we continue to refine the tool. Feel free to try it out and share your feedback!

== Key Features

* *xref:dev-services:index.adoc[Dev Services]*. First-class support for development and testing modes in Spring Boot applications. Dev Services automatically provisions external services using containers, making local development and testing easier, more productive, and more fun.
* *xref:opentelemetry:index.adoc[OpenTelemetry]*. Providing unified observability for Spring Boot applications, combining full support for OpenTelemetry and Micrometer for logs, metrics, and traces.
* *xref:observability:index.adoc[AI Observability]*. Supporting AI observability with pluggable semantic conventions, including xref:observability:semantic-conventions/opentelemetry.adoc[OpenTelemetry], xref:observability:semantic-conventions/openinference.adoc[OpenInference], xref:observability:semantic-conventions/openlit.adoc[OpenLit], xref:observability:semantic-conventions/openllmetry.adoc[OpenLLMetry], and xref:observability:semantic-conventions/langsmith.adoc[LangSmith], with integrations for platforms like xref:dev-services:phoenix.adoc[Arize Phoenix], xref:dev-services:lgtm.adoc[Grafana LGTM], and xref:dev-services:openlit.adoc[OpenLit].
* *xref:generative-ai:docling-document-reader.adoc[Docling]*. AI-powered document processing, enabling structured content extraction from PDFs, Office files, and HTML documents for use in AI pipelines.
* *xref:multitenancy:index.adoc[Multitenancy]*. Offering built-in support for multitenant applications, including web, data, security, and configuration aspects.
* *xref:kubernetes:index.adoc[Kubernetes]*. Simplifying the configuration and deployment of Spring Boot applications to Kubernetes, including the build of multi-architecture container images and support for the Service Binding specification.
