= Pulsar Dev Service
:service-name: pulsar

A service providing an https://pulsar.apache.org[Apache Pulsar] event streaming and messaging broker for development and testing purposes.

It works with Spring Boot libraries that support Apache Pulsar, including:

* https://spring.io/projects/spring-pulsar[Spring for Apache Pulsar]
* https://spring.io/projects/spring-cloud-stream[Spring Cloud Stream]
* https://spring.io/projects/spring-kafka[Spring Modulith]

== Configuring the Dev Service

You can configure the Dev Service via configuration properties.

|===
|Property |Default |Description

| `arconia.dev.services.pulsar.enabled`
| `true`
| Whether the dev service is enabled.

| `arconia.dev.services.pulsar.image-name`
| `apachepulsar/pulsar`
| Full name of the container image used in the dev service.

| `arconia.dev.services.pulsar.environment`
| `{}`
| Environment variables to set in the service.

| `arconia.dev.services.pulsar.shared`
| `dev-mode`
| When the dev service is shared across applications.
|===
