= Artemis Dev Service
:service-name: artemis

A service providing an https://activemq.apache.org/components/artemis/[ApacheMQ Artemis] event broker for development and testing purposes.

It works with Spring Boot libraries that support JMS and ApacheMQ Artemis, including:

* https://docs.spring.io/spring-framework/reference/integration/jms.html[Spring JMS]
* https://docs.spring.io/spring-boot/reference/messaging/jms.html#messaging.jms.artemis[Spring Boot Artemis]

== Configuring the Dev Service

You can configure the Dev Service via configuration properties.

|===
|Property |Default |Description

| `arconia.dev.services.{service-name}.enabled`
| `true`
| Whether the dev service is enabled.

| `arconia.dev.services.{service-name}.image-name`
| `apache/activemq-artemis`
| Full name of the container image used in the dev service.

| `arconia.dev.services.{service-name}.environment`
| `{}`
| Environment variables to set in the service.

| `arconia.dev.services.{service-name}.network-aliases`
| `[]`
| Network aliases to assign to the dev service container.

| `arconia.dev.services.{service-name}.port`
| `0`
| Fixed port for exposing the Artemis' TCP port to the host (CORE, MQTT, AMQP, HORNETQ, STOMP, OPENWIRE). When it's 0 (default), a random available port is assigned dynamically.

| `arconia.dev.services.{service-name}.shared`
| `true`
| Whether the dev service is shared among applications. Only applicable in dev mode.

| `arconia.dev.services.{service-name}.startup-timeout`
| `30s`
| Maximum waiting time for the service to start.

| `arconia.dev.services.{service-name}.management-console-port`
| `0`
| Fixed port for exposing the Artemis Management Console to the host. When it's 0 (default), a random available port is assigned dynamically.

| `arconia.dev.services.{service-name}.username`
| `artemis`
| Username for the Artemis administrator user.

| `arconia.dev.services.{service-name}.password`
| `artemis`
| Password for the Artemis administrator user.
|===
