= Light LDAP (LLDAP) Dev Service
:service-name: lldap

A service providing an https://ldap.com[LDAP] instance for development and testing purposes. It's based on https://github.com/lldap/lldap[LLDAP], which is "a lightweight authentication server that provides an opinionated, simplified LDAP interface for authentication".

It works with Spring Boot libraries that support LDAP, including:

* https://spring.io/projects/spring-data-ldap[Spring Data LDAP]

== Configuring the Dev Service

You can configure the Dev Service via configuration properties.

|===
|Property |Default |Description

| `arconia.dev.services.lldap.enabled`
| `true`
| Whether the dev service is enabled.

| `arconia.dev.services.lldap.image-name`
| `lldap/lldap`
| Full name of the container image used in the dev service.

| `arconia.dev.services.lldap.environment`
| `{}`
| Environment variables to set in the service. Example: `LLDAP_LDAP_BASE_DN: "dc=example,dc=com"`.

| `arconia.dev.services.lldap.shared`
| `never`
| When the dev service is shared across applications.
|===
