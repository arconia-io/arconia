= Light LDAP (LLDAP) Dev Service
:service-name: lldap

A service providing an https://ldap.com[LDAP] instance for development and testing purposes. It's based on https://github.com/lldap/lldap[LLDAP], which is "a lightweight authentication server that provides an opinionated, simplified LDAP interface for authentication".

It works with Spring Boot libraries that support LDAP, including:

* https://spring.io/projects/spring-data-ldap[Spring Data LDAP]

== Configuring the Dev Service

You can configure the Dev Service via configuration properties.

|===
|Property |Default |Description

| `arconia.dev.services.{service-name}.enabled`
| `true`
| Whether the dev service is enabled.

| `arconia.dev.services.{service-name}.image-name`
| `lldap/lldap`
| Full name of the container image used in the dev service.

| `arconia.dev.services.{service-name}.environment`
| `{}`
| Environment variables to set in the service.

| `arconia.dev.services.{service-name}.network-aliases`
| `[]`
| Network aliases to assign to the dev service container.

| `arconia.dev.services.{service-name}.port`
| `0`
| Fixed port for exposing the LDAP port to the host. When it's 0 (default), a random available port is assigned dynamically.

| `arconia.dev.services.{service-name}.shared`
| `false`
| Whether the dev service is shared among applications. Only applicable in dev mode.

| `arconia.dev.services.{service-name}.startup-timeout`
| `30s`
| Maximum waiting time for the service to start.

| `arconia.dev.services.{service-name}.management-console-port`
| `0`
| Fixed port for exposing the LLDAP Management Console to the host. When it's 0 (default), a random available port is assigned dynamically.
|===
