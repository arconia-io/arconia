= MongoDB Atlas Dev Service
:service-name: mongodb-atlas

A service providing a https://www.mongodb.com[MongoDB] database for development and testing purposes.

It works with Spring Boot libraries that support MongoDB Atlas including:

* https://spring.io/projects/spring-ai[Spring AI] (Memory, Vector Store)
* https://docs.spring.io/spring-ai/reference/api/vectordbs/mongodb.html[Spring AI] (Vector Store & Search)
* https://spring.io/projects/spring-data-mongodb[Spring Data MongoDB]
* https://spring.io/projects/spring-session[Spring Session]

== Configuring the Dev Service

You can configure the Dev Service via configuration properties.

|===
|Property |Default |Description

| `arconia.dev.services.mongodb-atlas.enabled`
| `true`
| Whether the dev service is enabled.

| `arconia.dev.services.mongodb-atlas.image-name`
| `mongodb/mongodb-atlas-local`
| Full name of the container image used in the dev service.

| `arconia.dev.services.mongodb-atlas.environment`
| `{}`
| Environment variables to set in the service. Example: `MONGO_INITDB_ROOT_USERNAME: "mongodb"`.

| `arconia.dev.services.mongodb-atlas.shared`
| `never`
| When the dev service is shared across applications.
|===
